비개발 1인 운영, 비용에 민감한 워크플로우에 맞춰 조사한 보완 도구 정리. 메인은 그대로 두고, 비용 없이 보강하는 쪽으로.

01 — Base

메인은 그대로

중심 도구를 자주 바꾸지 않는다. 익숙함도 생산성이다.

중심 도구

Core

메인은 Claude Code를 유지한다. 구독에 포함되어 있어 추가 비용이 들지 않고, 작업 흐름이 이미 여기에 맞춰져 있다. 보완 도구는 "메인을 대체"하려는 게 아니라 "빈 곳을 메우는" 역할로만 고른다.

기준 시점: 2026년 6월 조사. 가격·정책·인수합병은 자주 바뀌므로, 도입 직전 한 번 더 확인할 것.

02 — Add

비용 없이 보강

무료·오픈소스 위주로, 부담 없이 더한다.

보조 코딩 도구

Coding

Cline

오픈소스(무료). 내 API 키를 꽂아 쓰는 방식. 최우선 보조 후보.

Copilot Free

무료 등급. 코드 한 줄씩 자동완성(인라인)에 유용.

Codex

유지. 결과물을 다른 시선으로 교차감사하는 용도.

연결 확장 (MCP·커넥터)

Connect

AI가 내 자료에 직접 닿게 해주는 연결 도구들. 노션·피그마·구글(드라이브·캘린더·메일)을 연동하면, 따로 복사해 붙여넣지 않고도 맥락을 가져올 수 있다.

  • Notion 연동 — 업무 기록·문서
  • Figma 연동 — 디자인 자료
  • Google 커넥터 — 드라이브·캘린더·메일
