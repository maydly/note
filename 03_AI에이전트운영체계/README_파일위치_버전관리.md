# AI 에이전트 운영 체계 파일 위치 및 버전관리 규정

기준일: 2026-06-19

## 1. 공개 최신본

아래 파일을 공개 최신본으로 사용한다.

```text
03_AI에이전트운영체계/AI에이전트운영체계_작업장운영교육페이지.html
```

공개 URL:

```text
https://bucket0825.github.io/note/03_AI에이전트운영체계/
```

`index.html`은 위 공개 URL에서 최종본 HTML로 이동시키는 입구 파일이다.

## 2. 버전 보존본

보존본은 날짜별 버전 폴더에 둔다.

```text
03_AI에이전트운영체계/versions/2026-06-19/AI에이전트운영체계_작업장운영교육페이지_260619_1730_v3.0.html
```

## 3. 파일명 규정

- 최종본은 날짜와 버전을 붙이지 않는다.
- 버전 파일은 `대상_세부내용_유형_YYMMDD_HHMM_v버전.html` 형식으로 둔다.
- 이 페이지의 대상은 `AI에이전트운영체계`, 세부내용은 `작업장운영`, 유형은 `교육페이지`다.
- 기존 버전은 삭제하지 않고 `versions/YYYY-MM-DD/` 안에 보존한다.

## 4. 공개 전 검수 규정

- 로컬 절대경로, 계정명, 키, 비밀번호, 내부 명령어를 그대로 공개하지 않는다.
- 에이전트 구조와 운영 원리는 설명하되, 민감한 실행 환경 세부는 설명형으로 정리한다.
- 작업장 내부 규정과 프로젝트별 규정은 공개 가능한 수준으로만 요약한다.
- 수빈 프로젝트처럼 개인 기록 성격의 프로젝트는 원칙과 공개 주의만 설명하고 세부 기록은 노출하지 않는다.
